# Curiosity Webring

Welcome to the Curiosity Webring! This is a webring for builders at [Codédex](https://codedex.io) and beyond.

## How to Join

We have an automated process for joining the webring! 

1. Go to the [Issues tab](../../issues/new/choose).
2. Select **Join the Webring**.
3. Fill out the form with your ID, Display Name, and Website URL.
4. Submit the issue! Our automated bot will process your request and open a Pull Request to add you.
5. Once merged, you're officially part of the ring!

## Running Locally

If you want to run this project locally:

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) with your browser.
